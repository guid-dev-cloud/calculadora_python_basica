# calculadora_python_basica

Este projeto consiste em uma calculadora básica desenvolvida para rodar via terminal, permitindo operações aritméticas entre dois números.

Operações: Soma, Subtração, Multiplicação e Divisão.

Comandos Utilizados para criar e executar o script no ambiente Linux:

Bash

Criar o arquivo: nano calculadora.sh

Dar permissão de execução: chmod 744 calculadora.sh

Executar o script: ./calculadora.sh

Se o seu arquivo for .py, utilize python3 calculadora.py para executar
